Programme simple de menu recapitulant les bases du langage JS.

Pour l'utiliser vous allez avoir besoin de Node.js (afin d'excuté le code ne terminal) et de vs code. 

Pour ce faire : - télécharger node.js a cette adresse https://nodejs.org/
                - télécharger vs code a cette adresse https://code.visualstudio.com/

une fois dans vs code dans le main.js ouvrer le terminal et rentrez cette commande pour installer readline-sync qui un module JS pour l'affichage du code JS en terminal :

- npm install readline-sync

après quoi vous pouvez exécuté le code avec la commande : 

- node main.js

assuré d'avoir le terminal ouver dans le bon répertoire.
